◆ handleModeChange()

void handleModeChange ( )

Handles mode change.

Turns off all LEDs, blinks the specified LED according to the mode change, plays a beep sound if the next mode is out of range, limits the mode to 0-5, sets startMode5 flag if next mode is 5, updates the current mode string, and prints current mode string to Serial and OLED display.
