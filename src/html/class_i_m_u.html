◆ initializeIMU()

void IMU::initializeIMU ( )

Initialize the IMU sensor using SPI to not interfere with LoRA.

Note
If IMU is initialized using MISO and MOSI, this interferes with LoRa.

This method initializes the IMU sensor and checks for the status. If initialization fails, it prints an error message and enters an infinite loop.

Read all sensor values from the IMU.
