void MorseCodeController::translateToMorseCode ( String input)

Translates the input string into Morse code and calls the playMorse function.

For each character in the received string, translates the character to morse and plays it. The function takes into account the word and letter pause using Morse standards.

Parameters
inputThe input string to be translated.
